## Getting Started


```bash
npm run dev
```


## 사용 기술

- React
- Next.js
- TypeScript
- Recoil
- Styled-components

## 구현 기능
- 무한 스크롤 기능
- 상품 필터링 기능
  - 품절상품 기본으로 숨겨져 있으며, 필터로 표시 가능
  - 카테고리 필터기능
- 현재 조회된 데이터 범위 내 자동완성 기능을 포함한 검색기능
  - 디바운스 기능
- 상품 정보 표시
- 메타데이터 동적 생성 기능

## css 코드스타일
